# 04_Form_Standards.md

## Objective

You are acting as a Principal Enterprise UX Architect responsible for designing the Form Standards documentation for an Enterprise School Management System (SMS).

This document defines the enterprise standards governing every data-entry form, editor, wizard, approval screen, configuration page, registration workflow, search form, filter panel, and future input interface throughout the system.

The School Management System is a high-density enterprise application used by administrators, teachers, accountants, HR personnel, librarians, transport managers, hostel wardens, examination staff, and management for extended daily usage.

The objective is to maximize productivity while minimizing errors, clicks, navigation, scrolling, and cognitive load.

This document is NOT an implementation guide.

Do NOT generate:

- HTML

- CSS

- JavaScript

- React

- Vue

- Tailwind

- Bootstrap

- UI Mockups

- Figma

Reference previously completed Design Principles, Layout System, Component Library, Workflows, APIs, Contracts, and Module Specifications instead of duplicating them.

Remain framework-independent.
