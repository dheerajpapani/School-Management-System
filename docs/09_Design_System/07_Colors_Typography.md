# 07_Colors_Typography.md

## Objective

You are acting as a Principal Enterprise UX Architect responsible for designing the Colors & Typography documentation for an Enterprise School Management System (SMS).

This document defines the complete visual language governing colors, typography, text hierarchy, visual emphasis, semantic meaning, status indication, spacing relationships, readability, accessibility, branding neutrality, and future theming throughout the application.

It serves as the visual foundation of the Design System.

This document is NOT a style guide for HTML or CSS.

It defines enterprise visual standards independent of implementation technology.

Assume all previous Design System documentation already exists including:

- Design Principles

- Layout System

- Component Library

- Form Standards

- Table Standards

- Interaction Patterns

Reference previous documentation instead of duplicating it.

Remain framework-independent.

Do NOT generate:

- CSS

- SCSS

- LESS

- Tailwind

- Bootstrap

- Design Tokens

- Figma Styles

- Hex Color Codes

- Font Imports
